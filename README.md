# Netflix Data Cleaning & Analysis Project

## 📌 Objective
To clean and analyze a real-world Netflix dataset using Python and extract meaningful insights.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Data Cleaning Steps
- Handled missing values in `director`, `cast`, and `country`
- Converted `date_added` to datetime format
- Cleaned `duration` column by extracting numeric values
- Removed duplicate records
- Standardized dataset for analysis

## 📈 Analysis & Insights
- Majority of content on Netflix are Movies
- United States produces the highest number of shows
- Content production increased significantly after 2015
- Duration data was standardized for better analysis
